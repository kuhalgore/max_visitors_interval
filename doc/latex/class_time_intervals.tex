\doxysubsection{Detailed Description}
\doxylink{class_time_intervals}{Time\+Intervals} class. 

This class can read visiting times from an input file. The path of the input file is given as an argument to its \doxylink{class_time_intervals_a27016c92eac5a40f55a986c713e7c0c7}{read\+From\+File()} method. The input file is a text file, where every line is the entering and leaving time values of one visitor separated with commas. For example\+: 10\+:15,14\+:20 11\+:10,15\+:22 Visiting times can be in a random order and not sorted by entering or leaving time values. method \+: \doxylink{class_time_intervals_a27016c92eac5a40f55a986c713e7c0c7}{read\+From\+File(const std\+::string \& file\+Name)};

Using the data from the input file, this class can find the time period, when there were the most number of visitors and how many visitors were in the museum in that time period. method \+: \doxylink{class_time_intervals_ab0db3b45e15d8de01ad05e637254db3b}{find\+Max\+Interval\+And\+Count()};

This class can find the results found in the previous step to the standard output in the form $<$start time$>$-\/$<$end time$>$;$<$number of visitors$>$. For example\+: 11\+:10-\/14\+:20;2 method\+: \doxylink{class_time_intervals_a5822aad04a04c4d50356fac57eb2c6c9}{get\+Interval\+With\+Max\+Visitors()};
