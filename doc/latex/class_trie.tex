class Trie$<$ Val\+Type $>$}
Simple implemntation of a \doxylink{class_trie}{Trie} data structure We can use,
\begin{DoxyItemize}
\item \doxylink{class_trie_afb8da4677c55333e40ca6e3847b4ead3}{insert()} method to add a new entry
\item \doxylink{class_trie_a11c2053e7815c75b2c3bf5c2ad7fc4a4}{search()} method to find value corresponding the input key
\end{DoxyItemize}

We can use this \doxylink{class_trie}{Trie} wherein keys consists of only digits from \mbox{[}0 to 9\mbox{]} and key length is not too large. ( time complexity of \doxylink{class_trie_afb8da4677c55333e40ca6e3847b4ead3}{insert()} and \doxylink{class_trie_a11c2053e7815c75b2c3bf5c2ad7fc4a4}{search()} is O(n) where n is length of the input key ) This is ideal for our use case where in we want to find int value corresponding to its string reprenstaion of digits for example "{}00"{}-\/\texorpdfstring{$>$}{>} 00, "{}23"{}-\/-\/\texorpdfstring{$>$}{>}23 and "{}59"{}-\/-\/\texorpdfstring{$>$}{>}59, etc.
